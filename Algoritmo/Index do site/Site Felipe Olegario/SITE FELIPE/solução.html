Como funciona a nossa solução

O sensor escolhido foi o LM35 para a coleta de dados de temperatura no local do cultivo das microalgas. A coleta de dados é realizada de acordo com a necessidade do cliente, mas deixamos pré-programado a atualização dos dados a cada uma hora. Em nosso sistema o usuário vai entrar com o seu login e senha e após a validação dos dados o usuário tem acesso a relatórios, gráficos, históricos de acesso, área de suporte entre outras funções.

Nós então criamos uma solução para a parte de monitoramento da temperatura, com foco em baratear os custos desse processo e fazer com que seja atrativo o investimento nessa área. O processo de solução está organizado nas seguintes etapas: vamos mapear o local, criar o ambiente, instalar os nossos equipamentos e dar a interface para o usuário poder tomar decisões de acordo com as medições. As empresas de energia e combustível são nosso principal alvo de negócio.
